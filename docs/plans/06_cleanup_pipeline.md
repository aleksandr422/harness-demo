# Пайплайн 06 — Cleanup: Фоновая очистка (doc-gardening)

## Цель
Поддерживать актуальность документации и кодовой базы, фиксировать tech debt.

## Вход
- Содержимое `docs/`, `src/`, файл AGENTS.md.

## Шаги
1. Просканировать `docs/` на устаревшие куски (неактуальные спеки, планы).
2. Искать дубликаты и копипасту в `src/`.
3. Проверить, что AGENTS.md соответствует реальной структуре проекта.
4. Записать найденный tech debt в `docs/plans/tech_debt.md`.

## Выход
- Файл `docs/plans/tech_debt.md` со списком технического долга.

## Как запустить
Запускать периодически в фоне как doc-gardening между основными пайплайнами.
