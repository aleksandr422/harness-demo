# Пайплайн 03 — Code: ExecPlan → Код

## Цель
Реализовать следующую невыполненную задачу из плана.

## Вход
- Файл `docs/plans/execplan.md`.

## Шаги
1. Прочитать AGENTS.md и `docs/rules/architecture.md`.
2. Взять следующую невыполненную задачу из `docs/plans/execplan.md`.
3. Написать код в `src/` согласно описанию и критериям готовности.
4. Запустить `bash scripts/lint.sh`.
5. Если линтер ругается — исправить ошибки и запустить линтер заново.
6. Когда линтер зелёный — отметить задачу выполненной в `docs/plans/execplan.md`.

## Выход
- Новый/изменённый код в `src/`.
- Обновлённый чек-лист в `docs/plans/execplan.md`.

## Как запустить
Попросить агента выполнить пайплайн 03 для следующей задачи.
