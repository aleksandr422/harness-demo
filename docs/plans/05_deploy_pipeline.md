# Пайплайн 05 — Deploy: Merge & Deploy

## Цель
Зафиксировать готовый инкремент и подготовить релиз.

## Вход
- Прошедший review код в `src/` и тесты в `tests/`.

## Шаги
1. Запустить `bash scripts/lint.sh` — прогон должен быть зелёным.
2. Запустить `pytest` — все тесты должны проходить.
3. Запустить приложение локально и убедиться, что оно работает.
4. Сделать `git commit` с описательным сообщением.
5. Записать запись о релизе в `docs/plans/releases.md`.

## Выход
- Коммит с готовым инкрементом.
- Файл `docs/plans/releases.md` с историей релизов.

## Как запустить
Попросить агента выполнить пайплайн 05 после успешного review.
